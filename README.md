# Toll Plaza Performance Dashboard 

## Overview
This project presents an interactive Power BI dashboard that analyzes toll plaza operations across India, focusing on revenue, traffic volume, and vehicle category performance.

## Objectives
- Analyze total and average revenue generated by toll plazas  
- Understand traffic distribution across vehicle categories  
- Identify high-performing toll plazas and revenue trends over time  

## Key KPIs
- Total Revenue  
- Average Revenue per Vehicle  
- Number of Toll Plazas  
- Commercial Vehicle Share (%)  

## Key Insights
- Commercial vehicles contribute a significant share of total toll revenue  
- A small number of toll plazas generate a major portion of overall revenue  
- Monthly revenue trends indicate seasonal variations in traffic flow  

## Dashboard Features
- Interactive slicers for city-wise analysis  
- Revenue and traffic trends over time  
- Vehicle category distribution analysis  
- Geographic visualization of toll plaza performance  

## Tools & Technologies
- Power BI  
- DAX  
- Data Modeling  
- Data Visualization  

## Data Source
ETC transaction data sourced from official reports published by the Indian Highways Management Company Limited (IHMCL):  
https://ihmcl.co.in/etc-transaction-reports/

## How to Use
1. Download the `.pbix` file from this repository  
2. Open it using Power BI Desktop  
3. Use filters and slicers to explore toll plaza performance insights  

## Disclaimer
This project is created for learning and portfolio purposes. The analysis is based on publicly available data, and interpretations may not reflect official conclusions.
